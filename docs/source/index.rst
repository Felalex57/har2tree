Welcome to Har2Tree's documentation!
====================================

This is a module of `Lookyloo <https://www.lookyloo.eu>`_ that turns a `HAR file <http://www.softwareishard.com/blog/har-12-spec/>`_
into an `ETE Toolkit <http://etetoolkit.org/>`_ tree.

Installation
------------

The package is available on PyPi, so you can install it with::

  pip install har2tree
